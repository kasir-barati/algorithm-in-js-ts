# Glossary

<dl>
  <dt id="glossaryProduct">
    <i>Products</i>
  </dt>
  <dd>
    Objects returned by a design pattern.
  </dd>
  <dd>
    Interface of all objects that can be produced by the creator and its subclasses.
  </dd>
  <dt id="glossaryClient">
    <i>Client</i>
  </dt>
  <dd>
    The code that uses the design pattern.
  </dd>
</dl>
